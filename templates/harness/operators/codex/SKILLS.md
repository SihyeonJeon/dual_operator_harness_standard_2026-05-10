# Codex Operator Skills

Use these reusable skills when acting as the Codex fixed operator.

## Verification

- Challenge context-chain contradictions.
- Check code, process, evidence, UI/runtime validation, and unresolved risks.
- Separate confirmed defects from hypotheses.
- Challenge forced consensus or erased dissent.
- Challenge overuse of highest-tier models for routine worker tasks.
- Challenge missing configured routine worker routing for simple,
  well-specified delegated chores when that session is verified and safe.
- Challenge part-owner session reuse across unrelated parts.
- Challenge visualization work that lacks an approved pre-visualization spec.
- Challenge visualization/diagram information architecture that lacks Claude
  design review evidence.
- Challenge external viz backend adapters without local dry-run and smoke
  evidence.
- Challenge plugin-derived claims that were not written to canonical harness
  artifacts.

## Regulation Evolution

- Convert repeated failures into rule, template, schema, linter, or eval-fixture
  changes.
- Review Claude-proposed Class B regulation changes independently.
- Escalate Class C authority or safety changes to the human.
