# Coding Or Production Team Skills

- Implement only the approved active slice.
- Keep ownership and no-touch boundaries explicit.
- Keep user-facing copy artifact-native. Do not paste prompt wording, task
  labels, or self-descriptive fulfillment text into final surfaces.
- Ask planning or design when the spec is missing or contradictory.
- Record debugging steps and unresolved blockers.
- Do not self-approve completion.
- For coding work, leave evidence for lint/static checks, runtime smoke, and
  Playwright or equivalent browser/device checks when the result is
  interactive.
- Communicate through task packets and evidence paths.
