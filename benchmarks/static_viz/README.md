# Static Visualization Regression Guard

This guard verifies the default local visualization path generated by the kit.

It scaffolds a harness, runs `./init.sh`, exports events through the default
`local_file` backend, regenerates the static status report, and checks:

- verified local backend
- visualization policy boundary
- status HTML and JSON
- viz summary, JSON, and NDJSON
- no external network write
- event count consistency
- exported event schema allowlist
- redaction smoke for common secret and local path patterns
- private public-source marker scan
- source metadata for `events.jsonl`
- compiled-view authority in status JSON
- audit event for the viz export

This is not a hosted dashboard benchmark and not a UX quality benchmark. It
only proves that the generated harness can turn local canonical events into
sanitized local evidence files without network writes.

```sh
python3 benchmarks/static_viz/score.py --check-summary
```
